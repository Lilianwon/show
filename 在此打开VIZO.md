# 在 Cursor 右侧打开 VIZO

✅ **开发服务器已启动**，运行在 http://localhost:5173

## 方式一：快捷键（最快）

按 **`Cmd + Shift + V`** → VIZO 面板会直接在 Simple Browser 中打开。

**放到右侧**：打开后按 `Cmd + \` 或右键标签 → **Split Right**

## 方式二：运行任务

1. 按 **`Cmd + Shift + P`**
2. 输入 **`Tasks: Run Task`**
3. 选择 **`右侧打开 VIZO 面板`**

## 方式三：Simple Browser 手动输入

1. 按 **`Cmd + Shift + P`**
2. 输入 **`Simple Browser`**
3. 选择 **Simple Browser: Show**
4. 输入：**`http://localhost:5173`**

## 方式四：Live Preview 扩展

1. 安装扩展：搜索 **Live Preview**（Microsoft）
2. 按 `Cmd + Shift + P`，输入 **Live Preview: Show**
3. 选择 **Attach to Running Server**
4. 输入端口：**5173**

## 方式五：外部浏览器

在资源管理器中双击 **`点击打开VIZO.html`**，会在默认浏览器中打开 VIZO。
